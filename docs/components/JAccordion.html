JAccordion

Un contenedor que permite mostrar múltiples paneles de contenido expandible (`JAccordionPane`), útil para organizar información en espacios reducidos.

Ejemplo

Panel 1: Configuración General

Configuraciones básicas de la aplicación...

Panel 2: Avanzado 
JAccordion accordion = new JAccordion();

JAccordionPane pane1 = new JAccordionPane("Configuración General", new Label("Contenido..."));
JAccordionPane pane2 = new JAccordionPane("Avanzado", new Label("Opciones extra..."));

accordion.getPanes().addAll(pane1, pane2);
accordion.setExpandedPane(pane1); // Optional: Expand first one

Componentes Relacionados

JAccordionPane

Extiende `TitledPane` y se utiliza como hijo directo de `JAccordion`. Soporta animaciones suaves por defecto.

API Reference

Constructor Descripción
new JAccordion(TitledPane... panes) Crea un acordeón con los paneles dados.
new JAccordionPane(String title, Node content) Crea un panel individual con título y contenido.
