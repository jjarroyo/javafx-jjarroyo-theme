JAlert

Componente para mostrar mensajes de alerta contextuales con iconos, títulos y descripciones.

Ejemplos

Información

Este es un mensaje de información para el usuario.

Éxito

La operación se ha completado correctamente.

Error

Ha ocurrido un error al procesar la solicitud. 

JAlert infoAlert = new JAlert("Información", "Este es un mensaje de información...");
infoAlert.addClass("alert-primary");
infoAlert.setIcon(textIconP);

JAlert successAlert = new JAlert("Éxito", "La operación se ha completado correctamente.");
successAlert.addClass("alert-success");

// Set SVG Icon
successAlert.setIcon(new JIcon(JIcon.CHECK_CIRCLE));

// Enable Dismiss (X button)
successAlert.setDismissible(true);

API Reference

Método Descripción
new JAlert(String title, String desc) Constructor principal.
setIcon(JIcon icon) Establece un icono SVG a la izquierda del contenido.
setDismissible(boolean val) Habilita un botón de cierre (X) a la derecha que oculta la alerta al hacer clic.
addClass(String val) Método para añadir variantes de color: `alert-primary`, `alert-success`, `alert-danger`, etc.
