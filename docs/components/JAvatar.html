avatar.setSize(JAvatar.Size.MD);  // 40px (default)

API Reference

Método Descripción
new JAvatar(String initials) Constructor con iniciales directas.
new JAvatar(String first, String last) Auto-genera iniciales del nombre.
setSize(Size size) XS (24px), SM (32px), MD (40px), LG (48px), XL (64px).
setColor(String color) primary, success, danger, warning, info, slate.
setImage(Image img) Muestra una imagen circular en lugar de iniciales.
setStatus(Status status) ONLINE, AWAY, BUSY, OFFLINE, NONE.
setInitials(String initials) Cambia las iniciales dinámicamente.
new JAvatar.Group(JAvatar...) Grupo de avatares apilados.
group.setMax(int max) Máximo visible; extra muestra "+N".
