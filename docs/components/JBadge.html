JBadge

Etiquetas pequeñas para mostrar estados, contadores o categorías. Hereda de `javafx.scene.control.Label`.

Variantes y Colores

Primary Success Danger Warning Info Dark 
JBadge badge = new JBadge("New"); // Default is primary
JBadge successBadge = new JBadge("Completed", "badge-success");
JBadge dangerBadge = new JBadge("Error", "badge-danger");

// Badge with Icon
JBadge iconBadge = new JBadge("Settings", new JIcon(JIcon.SETTINGS));

Soporte para Iconos

Seguro Verificado 
JBadge badge = new JBadge("Verificado", "badge-light-success");
badge.setIcon(new JIcon(JIcon.CHECK));

Formas y Tamaños

Default Large 9 
// Large Badge
JBadge lgBadge = new JBadge("Large", "badge-lg");

// Circle Badge (Good for counters)
JBadge countBadge = new JBadge("5", "badge-circle");

API Reference

Método Descripción
new JBadge(String text, String... classes) Crea un badge con el texto y clases opcionales.
new JBadge(String text, JIcon icon) Crea un badge con texto y un icono.
setIcon(JIcon icon) Establece el icono del badge. Si es null, lo elimina.
addClass(String... classes) Añade clases CSS. Gestiona el reemplazo de la variante de color por defecto.
