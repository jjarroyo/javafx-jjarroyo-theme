JBreadcrumb

Navegación tipo migas de pan para indicar la ubicación actual del usuario dentro de la aplicación.

Ejemplo Básico

Inicio / Usuarios / Perfil 
JBreadcrumb bc = new JBreadcrumb();
bc.addItem("Inicio", () -> navigate("home"));
bc.addItem("Usuarios", () -> navigate("users"));
bc.addItem("Perfil"); // último = activo

Separadores

Home / Settings / Profile
Home › Settings › Profile
Home → Settings → Profile
Home • Settings • Profile 
bc.setSeparator("/");  // default
bc.setSeparator("›");
bc.setSeparator("→");
bc.setSeparator("•");

Colores del Item Activo

Inicio / Sección / primary
Inicio / Sección / success
Inicio / Sección / danger
Inicio / Sección / warning
Inicio / Sección / info 
bc.setActiveColor("primary");   // indigo (default)
bc.setActiveColor("success");   // green
bc.setActiveColor("danger");    // red
bc.setActiveColor("warning");   // amber
bc.setActiveColor("info");      // blue

Con Íconos

Dashboard / Usuarios / Jorge Arroyo 
JBreadcrumb bc = new JBreadcrumb();
bc.addItem("Dashboard", JIcon.DASHBOARD.view(), () -> nav("dash"))
  .addItem("Usuarios", JIcon.PERSON.view(), () -> nav("users"))
  .addItem("Jorge Arroyo", JIcon.USER.view());

API Reference

Método Descripción
addItem(String text, Runnable action) Agrega un item clickeable.
addItem(String text) Agrega el item activo (último).
addItem(String, Node icon, Runnable) Item con ícono y acción.
setPath(String... paths) Establece todo el path de una vez.
setSeparator(String sep) Separador entre items (default: "/").
setActiveColor(String color) Color del item activo: primary, success, danger, etc.
clear() Limpia todos los items.
