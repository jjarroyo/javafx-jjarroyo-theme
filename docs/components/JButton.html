JButton

Un botón mejorado que extiende la clase nativa `javafx.scene.control.Button` con estilos predefinidos y soporte para iconos SVG.

Variantes

El componente soporta múltiples variantes de color para diferentes contextos semánticos. 

// JavaFX Code
JButton btnPrimary = new JButton("Primary"); // Default is primary
JButton btnSecondary = new JButton("Secondary").addClass("btn-secondary");
JButton btnSuccess = new JButton("Success").addClass("btn-success");
JButton btnDanger = new JButton("Danger").addClass("btn-danger");
JButton btnWarning = new JButton("Warning").addClass("btn-warning");
JButton btnInfo = new JButton("Info").addClass("btn-info");
JButton btnDark = new JButton("Dark").addClass("btn-dark");

Variantes Light

Versiones más sutiles para acciones secundarias o estados menos prominentes. 

// Light Variants
JButton btnLight = new JButton("Light").addClass("btn-light-primary");
JButton btnLightSuccess = new JButton("Success").addClass("btn-light-success");
JButton btnDashed = new JButton("Add New").addClass("btn-dashed");

Tamaños

Disponible en tres tamaños: pequeño (sm), normal (base) y grande (lg). 

JButton btnSmall = new JButton("Small").addClass("btn-sm");
JButton btnDefault = new JButton("Default");
JButton btnLarge = new JButton("Large").addClass("btn-lg");

Iconos

Soporte integrado para iconos SVG mediante la clase `JIcon`. 

// Button with Icon
JIcon icon = new JIcon(JIcon.STAR);
JButton btnIcon = new JButton("With Icon", icon);

// Or set manually
btnIcon.setIcon(new JIcon(JIcon.USER));

API Reference

Método Descripción Parámetros
addClass(String... classes) Añade clases CSS y gestiona inteligentemente las variantes. String... styleClasses
setIcon(JIcon icon) Establece un icono SVG para el botón. JIcon icon
