JCard

Un contenedor flexible y extensible para agrupar contenido relacionado. Soporta cabeceras, pies de página, barras de herramientas y contenido colapsable.

Ejemplo Básico

Título de la Tarjeta

Subtítulo descriptivo aquí

Este es el cuerpo de la tarjeta. Puedes colocar cualquier nodo de JavaFX aquí. 

JCard card = new JCard("Título de la Tarjeta", "Subtítulo descriptivo aquí");

// Add Toolbar items
card.addToolbarItem(new JButton("Acción"));

// Set Body content
card.setBody(new Label("Este es el cuerpo de la tarjeta..."));

// Set Footer
card.setFooter(new JButton("Guardar"));

Características

Colapsable

Permite al usuario ocultar/mostrar el contenido del cuerpo.

card.setCollapsible(true);

Scrollable

Añade barras de desplazamiento automáticamente si el contenido excede el tamaño.

card.makeScrollable();

API Reference

Método Descripción
setTitle(String title) Establece el título principal de la tarjeta.
setSubtitle(String subtitle) Establece un subtítulo debajo del título principal.
addToolbarItem(Node node) Añade un nodo (botón, icono, etc.) a la zona derecha de la cabecera.
setBody(Node content) Establece el nodo de contenido principal. Reemplaza el contenido anterior.
setFooter(Node content) Establece el contenido del pie de página.
setCollapsible(boolean val) Añade un icono de chevron en la toolbar que permite colapsar el cuerpo.
