JChip

Etiquetas y chips para categorías, estados, filtros y tags con variantes, colores e íconos.

Variante Soft (default)

primary success danger warning info slate 
new JChip("primary").setColor("primary"); // default variant = SOFT
new JChip("success").setColor("success");

Variante Filled

primary success danger warning info slate 
new JChip("primary").setColor("primary").setVariant(JChip.Variant.FILLED);

Variante Outlined

primary success danger warning info slate 
new JChip("primary").setColor("primary").setVariant(JChip.Variant.OUTLINED);

Tamaños

Small Medium Large 
new JChip("Small").setChipSize(JChip.Size.SM);
new JChip("Medium").setChipSize(JChip.Size.MD);  // default
new JChip("Large").setChipSize(JChip.Size.LG);

Con Íconos

Activo Error Info 
new JChip("Activo", JIcon.CHECK.view()).setColor("success");
new JChip("Error", JIcon.ERROR.view()).setColor("danger");

Dismissible (Cerrables)

JavaScript Java Python 
JChip chip = new JChip("JavaScript");
chip.setColor("primary")
    .setDismissible(true)
    .setOnDismiss(() -> container.getChildren().remove(chip));

API Reference

Método Descripción
new JChip(String text) Constructor con texto.
new JChip(String text, Node icon) Constructor con texto e ícono.
setColor(String color) primary, success, danger, warning, info, slate.
setVariant(Variant v) SOFT (default), FILLED, OUTLINED.
setChipSize(Size s) SM, MD (default), LG.
setIcon(Node icon) Ícono a la izquierda del texto.
setDismissible(boolean) Habilita el botón ✕ de cerrar.
setOnDismiss(Runnable) Callback al cerrar el chip.
setText(String) / getText() Getter/setter del texto.
