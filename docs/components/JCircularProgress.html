JCircularProgress

Un indicador de progreso circular personalizable.

Ejemplo

75% 
JCircularProgress progress = new JCircularProgress(40, 8);
progress.setProgress(0.75);
progress.setColor("#3b82f6");

API Reference

Método Descripción
setProgress(double val) Establece el progreso (0.0 a 1.0).
setColor(String webColor) Color de la barra de progreso (ej: "#ff0000").
setTrackColor(String webColor) Color del fondo del círculo.
