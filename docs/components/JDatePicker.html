JDatePicker

Selector de fecha estilizado que se integra con el sistema de diseño Metronic.

Ejemplo 

JDatePicker datePicker = new JDatePicker();
datePicker.setValue(LocalDate.now());

// Can be used with form styles manually if needed
datePicker.getStyleClass().add("form-input");

Nota: `JDatePicker` aplica estilos al editor interno (TextField) para que coincida con otros inputs.

API Reference

Constructor Descripción
new JDatePicker() Crea un selector de fecha por defecto.
new JDatePicker(LocalDate date) Crea un selector con una fecha preseleccionada.
