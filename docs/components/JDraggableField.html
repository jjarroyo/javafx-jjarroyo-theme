JDraggableField

Campo arrastrable para usar en conjunto con JDesignCanvas. Permite drag & drop de campos de datos.

Ejemplo

Nombre
Email
Teléfono
Fecha 
new JDraggableField("nombre", "Nombre", JIcon.USER)
new JDraggableField("email", "Email", JIcon.EMAIL)
new JDraggableField("telefono", "Teléfono", JIcon.PHONE)
new JDraggableField("fecha", "Fecha", JIcon.CALENDAR)

API Reference

Método Descripción
new JDraggableField(String id, String label, JIcon icon) Constructor con id, texto y ícono.
getFieldId() Obtiene el identificador del campo.
getFieldLabel() Obtiene el texto del campo.

Nota: JDraggableField está diseñado para trabajar junto con JDesignCanvas. Los campos se arrastran hacia las áreas del canvas para asignarlos a las líneas de diseño.
