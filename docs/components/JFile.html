JFile

Componente versátil para carga de archivos. Soporta modos: Botón, Avatar, Cuadrado y Zona de Drop.

Modos

Avatar Mode

Click to upload

DropZone Mode

Arrastra archivos aquí o haz clic

PNG, JPG hasta 10MB 

// Avatar
JFile avatar = new JFile(JFile.Mode.AVATAR);
avatar.getValidExtensions().addAll(".jpg", ".png");

// DropZone
JFile drop = new JFile(JFile.Mode.DROP_ZONE);
drop.setMaxFileCount(5);

API Reference

Método Descripción
setMode(Mode mode) BUTTON, AVATAR, SQUARE, DROP_ZONE.
getSelectedFiles() ObservableList con los archivos seleccionados.
setMaxFileCount(int n) Define cuántos archivos se pueden subir (Default 1).
setMaxFileSize(long bytes) Tamaño máximo por archivo (Default 10MB).
setShowEdit(boolean b), setShowRemove(boolean b) Habilita botones superpuestos para editar/borrar en modos de imagen.
