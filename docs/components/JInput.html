JInput

Campo de entrada de texto estandarizado con soporte para validación, etiquetas y placeholders.

Uso Básico 

// Simple Input
JInput input = new JInput("Ingresa tu usuario");

// Valid Helper: Creates Label + Input boxed in a VBox
VBox field = new JInput().createWithLabel("Email", true);

Estados de Validación 

JInput successInput = new JInput();
successInput.setStatus("form-input-success");

JInput errorInput = new JInput();
errorInput.setStatus("form-input-danger");

JInput warningInput = new JInput();
warningInput.setStatus("form-input-warning");

API Reference

Método Descripción
createWithLabel(String label, boolean required) Crea y devuelve un VBox que contiene una etiqueta y este input. Útil para formularios rápidos.
setStatus(String styleClass) Aplica estilos de estado (éxito, error, advertencia) eliminando los estados anteriores.
