JInputGroup

Contenedor horizontal que fusiona visualmente inputs, botones y etiquetas, eliminando bordes intermedios y redondeando solo las esquinas exteriores.

Ejemplo

$ 
JInputGroup group = new JInputGroup();
group.add(new Label("$"), new JInput("0.00"));

// Search Example
JInputGroup searchGroup = new JInputGroup();
searchGroup.add(new JInput("Buscar..."), new JButton("Buscar"));

Funcionamiento

El componente detecta automáticamente la posición de sus hijos y aplica las clases CSS correspondientes (`input-group-left`, `input-group-center`, `input-group-right`) para gestionar los bordes y el radio de las esquinas.

API Reference

Método Descripción
add(Node... nodes) Añade componentes al grupo y actualiza sus estilos.
