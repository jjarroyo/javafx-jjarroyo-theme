JList

Wrapper para `ListView` que incluye paginación, scroll infinito y estilos de celda "Card" preconfigurados.

JList<User> list = new JList<>();

list.setCellFactory(lv -> new ListCell<>() {
    @Override
    protected void updateItem(User item, boolean empty) {
        super.updateItem(item, empty);
        if (empty || item == null) {
            setGraphic(null);
        } else {
            setGraphic(createCard(item)); // Your custom node
        }
    }
});

list.setOnScrollBottom(() -> loadMore());

API Reference

Método Descripción
setCellFactory(...) Configura el renderizado de celdas y aplica automáticamente la clase `j-list-card-cell` para estilos visuales de tarjeta.
setOnScrollBottom(Runnable r) Callback para carga de datos por scroll infinito.
