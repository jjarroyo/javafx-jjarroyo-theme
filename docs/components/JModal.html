JModal

Ventana modal superpuesta con animaciones de entrada/salida y soporte para diferentes tamaños y fondos.

Ejemplo

Título del Modal

Aquí va el contenido de tu modal. Puede ser cualquier nodo de JavaFX. 

JCard content = new JCard("Título", new Label("Contenido..."));
JModal modal = new JModal(content);

// Show
modal.show();

// Close
// modal.close(); is called automatically on backdrop click by default.

Tamaños

Soporta tamaños predefinidos: SMALL, MEDIUM (default), LARGE, FULL. 

modal.setSize(JModal.Size.LARGE);
modal.setSize(JModal.Size.FULL); // Almost fullscreen

API Reference

Método Descripción
show() Muestra el modal en el contenedor raíz `JJArroyo.getModalContainer()`.
close() Cierra el modal con animación.
setSize(Size size) Ajusta el ancho máximo del diálogo.
setCloseOnBackdropClick(boolean b) Habilita/deshabilita cerrar al hacer clic fuera del diálogo.
