JNumberInput

Input numérico con botones de incremento/decremento, límites min/max, step y soporte para decimales.

Ejemplo Básico 

JNumberInput input = new JNumberInput();
input.setValue(5);

Con Límites

Cantidad (0-10)

Edad (18-99) 

input.setValue(5).setMin(0).setMax(10);
input.setValue(25).setMin(18).setMax(99);

Step y Decimales

Step: 5

Precio (0.5) 

input.setStep(5);                         // incrementa de 5 en 5
input.setStep(0.5).setIntegerOnly(false); // decimales

API Reference

Método Descripción
setValue(double) Establece el valor.
getValue() Obtiene el valor actual.
setMin(double) Valor mínimo permitido.
setMax(double) Valor máximo permitido.
setStep(double) Incremento por clic (default: 1).
setIntegerOnly(boolean) Solo enteros (default: true).
setOnValueChanged(Consumer) Callback cuando cambia el valor.
setPlaceholder(String) Texto placeholder del campo.
setDisabled(boolean) Deshabilita el componente.
getTextField() Acceso al TextField interno.
