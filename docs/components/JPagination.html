JPagination

Control de paginación numérico con soporte para navegación anterior/siguiente y truncamiento inteligente.

Ejemplo

... 
JPagination pagination = new JPagination();

// Set Totals
pagination.totalPagesProperty().set(10);
pagination.currentPageProperty().set(2);

// Listen for changes
pagination.setOnPageChange(() -> {
    int page = pagination.currentPageProperty().get();
    System.out.println("Cargar página: " + page);
    // loadData(page);
});

Lógica de Visualización

El componente calcula automáticamente qué botones mostrar. Si hay muchas páginas, muestra la primera, la última, la actual y sus vecinas, colapsando el resto con puntos suspensivos ("...").

API Reference

Propiedad/Método Descripción
currentPageProperty() IntegerProperty para la página actual (1-based).
totalPagesProperty() IntegerProperty para el número total de páginas.
setOnPageChange(Runnable r) Registra un callback que se invoca cuando el usuario cambia de página.
