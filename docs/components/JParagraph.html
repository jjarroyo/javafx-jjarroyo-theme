JParagraph

Un componente de texto optimizado para párrafos extensos y contenido HTML básico. Extiende `javafx.scene.control.Label`.

Ejemplo

Este es un párrafo centrado que demuestra cómo JParagraph maneja el ajuste de texto automáticamente por defecto.

Nota: JParagraph soporta alineación justificada y conversión básica de etiquetas HTML como <br> y <p> a saltos de línea planos para su uso en controles JavaFX estándar. 

JParagraph p = new JParagraph("Texto largo...");
p.setJustify();
p.setHtmlText("Línea 1
Línea 2");

API Reference

Método Descripción
text(String text) Establece el texto (Fluent API).
width(double w) Establece el ancho pref/min/max (Fluent API).
setHtmlText(String html) Convierte etiquetas basicas HTML a saltos de línea.
setAlignCenter/Right/Left/Justify() Configura la alineación del texto.
addClass(String... classes) Añade clases CSS.
