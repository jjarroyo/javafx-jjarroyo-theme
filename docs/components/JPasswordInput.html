JPasswordInput

Campo de contraseña seguro con funcionalidad de "mostrar/ocultar" integrada.

Ejemplo 

JPasswordInput password = new JPasswordInput("Ingrese contraseña");

// With Label
VBox field = new JPasswordInput().createWithLabel("Contraseña", true);

Características

  • Alternar Visibilidad: Incluye un botón de ojo a la derecha para mostrar/ocultar los caracteres.
  • Sincronización: Mantiene el texto sincronizado entre el campo de contraseña y el de texto visible internamente.
  • Integración: Diseñado para encajar perfectamente con otros inputs como `JInput`.

API Reference

Método Descripción
createWithLabel(String label, boolean required) Crea un contenedor con etiqueta y campo de contraseña.
new JPasswordInput(String prompt) Crea el componente con un texto de marcador (placeholder).
