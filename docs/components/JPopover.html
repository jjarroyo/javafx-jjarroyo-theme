JPopover

Pop-up ligero con flecha indicadora, ideal para tooltips ricos, menús contextuales o confirmaciones.

Ejemplo

Popover Title

Información adicional detallada aquí. 

JPopover popover = new JPopover();
popover.setTitle("Detalles");
popover.setBody("Este es un popover informativo.");

button.setOnAction(e -> {
    popover.show(button);
});

Modo Confirmación 

popover.setBody("¿Estás seguro de eliminar este ítem?");
popover.setConfirmationMode("Sí, eliminar", "No", () -> {
    deleteItem();
});

API Reference

Método Descripción
show(Node target) Muestra el popover anclado al nodo objetivo.
setPosition(Position pos) TOP, BOTTOM, LEFT, RIGHT.
setTheme(Theme t) LIGHT o DARK.
setConfirmMode(...) Configura botones de acción para flujos de confirmación.
