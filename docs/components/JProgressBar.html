JProgressBar

Barra de progreso multi-estilo para visualizar el avance de tareas.

Ejemplo

Primario
Éxito 
JProgressBar pb = new JProgressBar(0.45); // 45%

JProgressBar successPb = new JProgressBar(0.70);
successPb.setProgressStyle(JProgressBar.ProgressStyle.SUCCESS);

Tamaños y Estilos 

// Sizes
pb.setSize(JProgressBar.ProgressSize.SM);
pb.setSize(JProgressBar.ProgressSize.LG);

// Styles (Colors)
pb.setProgressStyle(JProgressBar.ProgressStyle.DANGER);
pb.setProgressStyle(JProgressBar.ProgressStyle.WARNING);
pb.setProgressStyle(JProgressBar.ProgressStyle.INFO);

API Reference

Método Descripción
setProgressStyle(ProgressStyle style) Cambia el color de la barra (PRIMARY, SUCCESS, DANGER, etc.).
setSize(ProgressSize size) Ajusta la altura de la barra (SM, MD, LG).
