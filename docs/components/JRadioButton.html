JRadioButton

Botón de selección única dentro de un grupo, con soporte para variantes de color y tamaño.

Ejemplo 

ToggleGroup group = new ToggleGroup();

JRadioButton rb1 = new JRadioButton("Opción 1");
rb1.setToggleGroup(group);
rb1.setSelected(true);

JRadioButton rb2 = new JRadioButton("Opción 2");
rb2.setToggleGroup(group);
rb2.setColorStyle(JRadioButton.RadioStyle.SUCCESS);

JRadioButton rb3 = new JRadioButton("Opción 3");
rb3.setToggleGroup(group);
rb3.setColorStyle(JRadioButton.RadioStyle.DANGER);

Tamaños 

JRadioButton small = new JRadioButton("Small");
small.setSize(JRadioButton.RadioSize.SMALL);

JRadioButton large = new JRadioButton("Large");
large.setSize(JRadioButton.RadioSize.LARGE);

API Reference

Método Descripción
setColorStyle(RadioStyle style) Cambia el color del radio button.
setSize(RadioSize size) Cambia el tamaño del radio button.
setToggleGroup(ToggleGroup group) Asigna el grupo para comportamiento de selección única.
