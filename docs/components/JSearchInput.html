JSearchInput

Campo de búsqueda con ícono, botón de borrar (clear) y soporte para autocompletado con popup.

Búsqueda Básica 

JSearchInput search = new JSearchInput();
search.setPromptText("Buscar por nombre...");
search.setOnSearch(query -> System.out.println("Resultados para: " + query));

API Reference

Método Descripción
setPromptText(String) Placeholder del TextField.
getText() / setText() Valor actual de la búsqueda.
setClearable(boolean) Ocultar/mostrar la ✕ para borrar (default: true).
setSuggestions(List) Asigna lista estática (filtra por contains).
setSuggestionProvider(Function) Retorna sugerencias dinámicas según texto.
setOnSearch(Consumer) Callback al presionar ENTER o seleccionar item.
getTextField() Acceso directo al control base.
