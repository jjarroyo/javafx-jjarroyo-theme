JSelect

Un componente de selección avanzado (Dropdown/ComboBox) con soporte para búsqueda, selección múltiple y carga dinámica.

Ejemplo Básico 

JSelect<String> countrySelect = new JSelect<>();
countrySelect.getItems().addAll("USA", "Canada", "Mexico", "Spain");

// Enable Search
countrySelect.setSearchable(true);

Selección Múltiple 

JSelect<String> tagSelect = new JSelect<>();
tagSelect.getItems().addAll("Design", "Development", "Marketing");
tagSelect.setMultiple(true);

// Get Selected Items
ObservableList<String> selected = tagSelect.getSelectedItems();

API Reference

Método Descripción
setMultiple(boolean val) Habilita/deshabilita la selección múltiple.
setSearchable(boolean val) Muestra un campo de búsqueda en la parte superior del popup.
getItems() Devuelve la lista observable de elementos.
setPlaceholder(String placeholder) Establece el texto de sugerencia cuando no hay nada seleccionado.
getSelectedItems() Retorna la lista observable de elementos seleccionados (Modo múltiple).
setCellFactory(Callback factory) Permite personalizar la apariencia de los elementos en la lista.
setConverter(Function<T, String> c) Define cómo convertir los objetos a texto para mostrar en la interfaz.
