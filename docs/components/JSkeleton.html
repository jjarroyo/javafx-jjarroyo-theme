JSkeleton

Placeholders animados de carga (shimmer/pulse) para indicar contenido que se está cargando.

Líneas de Texto 

JSkeleton.text(3); // 3 líneas animadas

Avatar + Texto (User Row) 

JSkeleton.userRow(); // avatar 40px + 2 líneas

Card Skeleton 

JSkeleton.card(); // imagen + título + 2 líneas

Formas Personalizadas 

JSkeleton.rect(80, 80);   // rectángulo
JSkeleton.rect(120, 40);  // barra
JSkeleton.avatar(48);     // círculo 48px
JSkeleton.avatar();       // círculo 40px (default)

Tabla 

JSkeleton.table(5, 4); // 5 filas, 4 columnas

API Reference

Método Descripción
JSkeleton.line(double width) Línea individual animada.
JSkeleton.text(int lines) Grupo de líneas de texto.
JSkeleton.rect(w, h) Rectángulo con esquinas redondeadas.
JSkeleton.avatar(double size) Círculo (default 40px).
JSkeleton.card() Card con imagen + texto.
JSkeleton.userRow() Avatar + 2 líneas de texto.
JSkeleton.table(rows, cols) Tabla de skeleton.
stop() Detiene la animación.
