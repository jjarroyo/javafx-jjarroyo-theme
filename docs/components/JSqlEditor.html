Atajos de Teclado

Ctrl+Shift+F Formatear SQL
Ctrl+Space Autocompletar
Ctrl+/ Comentar línea
Ctrl+D Duplicar línea
Ctrl+Shift+K Eliminar línea
Tab Indentar / Aceptar autocompletado

API Reference

Método Descripción
getSql() / setSql(String) Obtiene/establece el contenido SQL.
getText() / setText(String) Alias de getSql/setSql.
clear() Limpia el contenido del editor.
formatSql() Formatea el SQL con indentación.
setReadOnly(boolean) Modo solo lectura.
setDarkMode(boolean) Activa/desactiva el tema oscuro.
toggleDarkMode() Alterna entre tema claro y oscuro.
getSqlErrors() Lista de errores de validación SQL.
copyToClipboard() Copia el contenido al portapapeles.
focusEditor() Da foco al editor.
dispose() Libera subscripciones reactivas.
