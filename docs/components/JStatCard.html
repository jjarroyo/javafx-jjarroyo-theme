JStatCard

Un componente que muestra una tarjeta con métricas e indicadores de rendimiento clave (KPIs), el cual integra una cuenta regresiva para presentar sus números.

Uso Básico

El uso más simple es una tarjeta con título y un número asignado estáticamente, sin tendencia adicional. Por defecto animará en 1000 milisegundos desde cero al valor que determines. 

JStatCard stat = new JStatCard();
stat.setTitle("Usuarios Activos");
stat.setValue(2340);

Con Prefijos, Sufijos y Decimales

Puedes personalizar la forma en que se muestra el número, estableciendo cuántos decimales debe de mostrar usando setDecimalPlaces(). 

JStatCard revenueCard = new JStatCard();
revenueCard.setTitle("Ingresos Mensuales");
// Agregar símbolo de $ y moneda, para quedar ej: $15,000.55 MXN
revenueCard.setPrefix("$");
revenueCard.setSuffix(" MXN");
revenueCard.setDecimalPlaces(2);
revenueCard.setValue(15000.55);

Añadiendo Íconos y Tendencia (Trend)

Para mostrar si el KPI fue "bueno o malo" en comparación de un tiempo anterior, podemos asignar iconos en combinacion con colores y descripciones de las tendencias en positivo y negativo. 

JStatCard bounceCard = new JStatCard();
bounceCard.setTitle("Tasa de Rebote");
bounceCard.setSuffix("%");
bounceCard.setDecimalPlaces(1);
bounceCard.setValue(42.3, 2000); // Podemos cambiar la duración de la animación! Aqui: 2000ms.

// Asignamos un icono del Enum JIcon, y la clase de color 'danger'
bounceCard.setIcon(JIcon.CLOSE.view(), "danger");

// Texto de comparación o tendencia en rojo (isPositive = false)
bounceCard.setTrend("-2.1% que ayer", false);

API Methods

Method Description
setTitle(String) Cambia el título superior de la métrica (ej. "Total Orders").
setValue(double value)
setValue(double value, int durationMs) 		Establece el valor. Anima desde el valor en el que estaba anteriormente al nuevo de forma dinámica. Se le puede asignar un tiempo distinto que el de por defecto (1000m).
setIcon(Node icon, String colorClass) Muestra un icono en la zona derecha superior incrustado dentro de un circulo contenedor. Se recomienda colores como: `primary`, `success`, `danger`, `warning`, `info`...
setTrend(String text, boolean isPositive) Agrega el pie de página con el resultado de alguna comparación o texto que explique una tendencia. `isPositive` pondrá el texto en formato Success (Verde) o en formato Danger (Rojo).
setPrefix(String prefix)
setSuffix(String suffix) 		Strings de texto que irán concatenados justamente antes y después del número formateado. Por ejemplo: `$` para un prefix.
setDecimalPlaces(int limit) Cuantos decimales van a imprimirse junto al valor durante y al final de la animación de dicho número.
