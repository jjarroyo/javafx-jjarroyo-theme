JStepper

Componente de pasos secuenciales o asistente (wizard) con validación, progreso y navegación integrada.

Ejemplo Básico (Horizontal)

Account
Basic details
2
Profile
Personal info
3
Verification
ID Upload 
JStepper stepper = new JStepper();

// Crear contenidos
VBox step1 = new VBox(new Label("Email..."));
VBox step2 = new VBox(new Label("Nombres..."));

// Agregar pasos (Título, Subtítulo, Contenido)
stepper.addStep("Account", "Basic details", step1);
stepper.addStep("Profile", "Personal info", step2);

// Capturar eventos y lógica adicional
stepper.setOnStepChange((oldIndex, newIndex) -> {
    System.out.println("De paso " + oldIndex + " a " + newIndex);
});

stepper.setOnFinish(() -> {
    guardarYEnviarDatos();
});

// Personalizar botones si se necesita:
stepper.getBackButton().setText("Atrás");
stepper.getNextButton().setText("Siguiente");
stepper.getFinishButton().setText("Enviar");

API Reference

Método Descripción
addStep(String title, String subtitle, Node content) Añade un nuevo paso en la secuencia (el subtítulo es opcional, puede ser nulo o vacío).
setCurrentStepIndex(int index) Mueve el stepper programáticamente al índice especificado (0-based).
setOnStepChange(BiConsumer<Integer, Integer>) Callback al cambiar de paso. Provee `(índiceAntiguo, índiceNuevo)`. Ideal para validaciones.
setOnFinish(Runnable) Acción a realizar cuando se hace clic en el botón final (Finish).
getBackButton() / getNextButton() / getFinishButton() Permite acceder a los botones para cambiarles el texto o clases CSS.
