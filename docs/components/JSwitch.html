JSwitch

Un control de alternancia (toggle) que extiende `JCheckBox` pero con una apariencia de interruptor moderno.

Ejemplo

Notificaciones (Activado)
Modo Avión (Desactivado) 
JSwitch switchBtn = new JSwitch("Notificaciones");
switchBtn.setSelected(true);

// Listen for changes
switchBtn.selectedProperty().addListener((obs, oldVal, newVal) -> {
    System.out.println("Switch is now: " + newVal);
});

Herencia: Dado que `JSwitch` extiende de `JCheckBox`, hereda todas sus propiedades y métodos, pero se renderiza visualmente como un interruptor deslizante gracias a la clase CSS `j-switch`.

API Reference

Constructor Descripción
new JSwitch() Crea un switch sin texto.
new JSwitch(String text) Crea un switch con una etiqueta de texto.
setSelected(boolean value) Define el estado del interruptor (activado/desactivado).
