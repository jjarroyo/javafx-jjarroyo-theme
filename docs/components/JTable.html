JTable

Un wrapper para `TableView` que incluye paginación integrada (`JPagination`) y soporte para Infinite Scroll.

Ejemplo

Nombre Rol Estado
Juan Pérez Admin Activo
María García Editor Inactivo
Anterior Siguiente

Mostrando page 1 

JTable<User> table = new JTable<>();

// Add Columns
table.addColumn("Nombre", "name"); // Uses PropertyValueFactory("name")
table.addColumn("Rol", "role");

// Set Items
table.setItems(usersObservableList);

// Pagination Access
table.getPagination().totalPagesProperty().set(5);

// Infinite Scroll
table.setOnScrollBottom(() -> loadMoreData());

API Reference

Método Descripción
addColumn(String title, String property) Helper rápido para añadir columnas de texto simple.
setItemsPerPage(int items) Establece la cantidad de elementos a mostrar por página.
setPaginationVisible(boolean visible) Muestra u oculta los controles de paginación.
addItem(T item) Añade un elemento a la lista total de la tabla.
getLastTable() Retorna la instancia interna de `TableView` para configuración avanzada.
setOnScrollBottom(Runnable r) Habilita el scroll infinito y ejecuta la acción al llegar al final.
getPagination() Retorna el componente `JPagination` asociado.
