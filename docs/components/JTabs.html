JTabs

Sistema de pestañas flexible que soporta diferentes estilos (Línea, Pastilla) y orientaciones (Arriba, Abajo, Izquierda, Derecha).

Estilo Por Defecto (Line)

Perfil
Cuenta
Notificaciones
Contenido del Perfil... 
JTabs tabs = new JTabs();

tabs.getTabs().addAll(
    new JTab("Perfil", new Label("Contenido del Perfil...")),
    new JTab("Cuenta", new Label("Configuración de Cuenta...")),
    new JTab("Notificaciones", new Label("Preferencias..."))
);

Estilo Pills

Mensual
Anual
tabs.setTabStyle(JTabs.TabStyle.PILLS);

Orientación

Las pestañas pueden colocarse en los 4 lados. 

tabs.setSide(Side.LEFT);
// tabs.setSide(Side.RIGHT);
// tabs.setSide(Side.BOTTOM);

API Reference

Método Descripción
getTabs().add(JTab tab) Añade una nueva pestaña.
setTabStyle(TabStyle style) Cambia el estilo visual (LINE, PILLS).
setSide(Side side) Cambia la posición de los encabezados (TOP, BOTTOM, LEFT, RIGHT).
