JTextArea

Campo de entrada de texto multilínea con soporte para validación y etiquetas, consistente con `JInput`.

Ejemplo 

JTextArea comments = new JTextArea("Escribe tus comentarios aquí...");

// With Label and Required mark
VBox descriptionField = new JTextArea().createWithLabel("Descripción", true);

Validación 

JTextArea input = new JTextArea();
input.setStatus("form-input-danger"); // Red border
input.setStatus("form-input-success"); // Green border

API Reference

Método Descripción
createWithLabel(String label, boolean required) Crea un VBox con una etiqueta y el área de texto.
setStatus(String styleClass) Aplica estilos de validación (success, danger, warning).
