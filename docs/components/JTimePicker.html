JTimePicker

v1.1

Selector de hora compacto HH:mm con spinners ▲▼, escritura directa en el campo, validación automática y soporte de teclas ↑↓.

Vista Previa Interactiva

Predeterminado

:

Puedes hacer clic y escribir el número directamente

Compacto (styleClass compact)

: 
// Uso básico
JTimePicker picker = new JTimePicker();
JTimePicker picker = new JTimePicker("08:30");

// Leer el valor actual
String hora = picker.getValue(); // → "08:30"

// Establecer un valor desde código
picker.setValue("14:00");

// Binding bidireccional con ViewModel
picker.valueProperty().bindBidirectional(viewModel.formTimeProperty());

// Escuchar cambios
picker.valueProperty().addListener((obs, oldVal, newVal) -> {
    System.out.println("Hora seleccionada: " + newVal);
});

Modos de interacción

🖱️

Botones ▲▼

Horas: incrementa/decrementa de 1 en 1 (wrap 0–23).
Minutos: incrementa/decrementa de 1 en 1 (wrap 0–59).

⌨️

Escritura directa

Haz clic en el campo y escribe el número. Solo se aceptan dígitos. Al salir del campo (blur) o presionar Enter, el valor se valida y se aplica el rango correcto (0–23 / 0–59).

↑↓

Teclas del teclado

Con el campo enfocado, presiona ↑ para incrementar y ↓ para decrementar. Equivalente a usar los botones con el mouse.

Validación automática

Si el usuario escribe 99 en horas, al salir del campo se corregirá automáticamente a 23. Si deja el campo vacío, se restaura el último valor válido.

API Reference

Constructores

Constructor Descripción
new JTimePicker() Crea el componente con valor inicial "00:00".
new JTimePicker(String initialTime) Crea el componente con una hora preestablecida en formato "HH:mm". Ej: "08:30", "14:00".

Métodos

Método Retorna Descripción
getValue() String Retorna el valor actual en formato "HH:mm".
setValue(String v) void Establece la hora desde código. El formato debe ser "HH:mm".
valueProperty() StringProperty Property observable del valor. Úsala para binding bidireccional o para escuchar cambios.
getHours() int Retorna la hora actual como entero (0–23).
getMinutes() int Retorna los minutos actuales como entero (0–59).
getHourField() TextField Expone el TextField interno de las horas. Útil para estilos extra.
getMinuteField() TextField Expone el TextField interno de los minutos. Útil para estilos extra.

Ejemplo con ViewModel (MVVM) 

// En la Vista (View)
JTimePicker timePicker = new JTimePicker("09:00");

// Binding bidireccional → el campo y el ViewModel se sincronizan solos
timePicker.valueProperty().bindBidirectional(viewModel.formTimeProperty());

// El usuario puede escribir "14" en horas o usar ▲▼ — ambos actualizan el ViewModel

// ─────────────────────────────────────────────────
// En el ViewModel
private final StringProperty formTime = new SimpleStringProperty("09:00");

public StringProperty formTimeProperty() { return formTime; }
public String getFormTime()              { return formTime.get(); }
