JTimeline

Un componente para mostrar eventos, historial o pasos en un formato de línea de tiempo con orientación vertical u horizontal.

Uso Básico

System Update
10:00 AM
The system was updated to version 2.1.0.
Database Backup
11:30 AM
Automated backup completed without errors. 
JTimeline timeline = new JTimeline();

JTimeline.Event e1 = new JTimeline.Event("System Update", "The system was updated to version 2.1.0.", "10:00 AM");
e1.setColorClass("primary"); // Punto y línea azules

JTimeline.Event e2 = new JTimeline.Event("Database Backup", "Automated backup completed without errors.", "11:30 AM");
e2.setColorClass("success"); // Punto verde

timeline.addEvent(e1);
timeline.addEvent(e2);

Iconos y Contenido Personalizado

Puedes asignar un nodo custom (`Node`) como icono para cada paso, en lugar del punto por defecto. También es posible agregar un visor o componente custom debajo de la descripción. 

JTimeline.Event eFile = new JTimeline.Event("Archivo Subido", "ReporteMensual.pdf", "Hoy");
eFile.setIcon(JIcon.FILE.view()); // Icono del tema
eFile.setColorClass("info");  // Color de resaltado

// Contenido inyectado
VBox customReceipt = new VBox(new Label("Visor de archivo..."));
eFile.setCustomContent(customReceipt);

timeline.addEvent(eFile);

Orientación Horizontal

El `JTimeline` soporta disposición horizontal con `.setOrientation(Orientation.HORIZONTAL)`. 

JTimeline timelineH = new JTimeline();
timelineH.setOrientation(Orientation.HORIZONTAL);

timelineH.addEvent("Phase 1", "Planning", "Jan 2024");
timelineH.addEvent("Phase 2", "Development", "Feb 2024");
timelineH.addEvent("Phase 3", "Testing", "Apr 2024");
timelineH.addEvent("Phase 4", "Launch", "Jun 2024");

Propiedades y Métodos (`JTimeline`)

Método Descripción
setOrientation(Orientation) Define si la línea de tiempo es `VERTICAL` (por defecto) o `HORIZONTAL`.
addEvent(...) Agrega un `Event` (soporta sobrecarga básica y avanzada por objeto).
getEvents() Obtiene la lista observable de eventos (`ObservableList<Event>`).
