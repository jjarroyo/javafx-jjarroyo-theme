JToast

Notificaciones flotantes no bloqueantes que aparecen temporalmente para informar al usuario.

Ejemplos

Guardado

Los cambios se han guardado correctamente.

Error de Conexión

No se pudo conectar con el servidor. 

// Simple Toast
JToast.show(getWindow(), "Operación completada", JToast.Type.SUCCESS, JToast.Position.TOP_RIGHT, 3000);

// Toast with Title
JToast.show(getWindow(), "Error", "No se pudo guardar el archivo", JToast.Type.DANGER, JToast.Position.BOTTOM_RIGHT, 5000);

API Reference

Método Estático Descripción
show(Window owner, String msg, Type type, Position pos, int duration) Muestra un toast simple.
show(Window owner, String title, String msg, Type type, Position pos, int duration) Muestra un toast con título y mensaje.

Enums:

  • Type: DEFAULT, SUCCESS, DANGER, WARNING, INFO
  • Position: TOP_RIGHT, TOP_LEFT, BOTTOM_RIGHT, BOTTOM_LEFT, TOP_CENTER, BOTTOM_CENTER
