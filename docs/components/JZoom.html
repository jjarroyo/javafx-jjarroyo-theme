JZoom

Control de zoom con botones + y - para ajustar el nivel de zoom de contenido.

Ejemplo

100%
Texto de ejemplo 
JZoom zoom = new JZoom();
zoom.setOnZoomIn(() -> {
    // Incrementar zoom
});
zoom.setOnZoomOut(() -> {
    // Decrementar zoom
});

API Reference

Método Descripción
setOnZoomIn(Runnable) Callback al presionar el botón +.
setOnZoomOut(Runnable) Callback al presionar el botón −.
