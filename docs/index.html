Bienvenido a la Documentación

Esta librería proporciona un conjunto de componentes UI para JavaFX inspirados en Metronic y Tailwind CSS. Los componentes están diseñados para ser modernos, flexibles y fáciles de usar.

Diseño Moderno

Estilos limpios y consistentes basados en utilidades CSS modernas.

Fácil Integración

Componentes listos para usar que extienden los controles nativos de JavaFX.

Componentes Destacados

Botones

Botones con variantes de color, tamaño y estilos.

Inputs

Campos de texto modernos con estados y validación.

Tarjetas

Contenedores flexibles para agrupar contenido relacionado.
